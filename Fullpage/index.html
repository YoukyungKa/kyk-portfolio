PROJECT 03

DELICIOUSA RESTAURANT

Responsive Website

POINT COLOR

'DELICIOUSA' 레스토랑 홍보 반응형 웹 페이지입니다.
반응형으로 제작해 사용자들의 각자 다른 해상도에서도
원활한 이용이 가능하게 했습니다.
레스토랑 웹의 특성을 강조하여 식욕을 돋구는 색상인
붉은 색을 사용하였습니다. 전체배경은 음식이 돋보이
게 하기 위해 하얀색을 사용했습니다.

서브 페이지에서는 레스토랑의 뉴스와 이벤트 영상 등을
전달하기 용이하게 제작했습니다.

PROJECT 04

LIGHTING STORE

Shopping Mall

POINT COLOR

Lighting Store는 사용자의 취향에 맞게 구매할 수 있도
록 다양한 조명을 판매하는 웹사이트입니다.
스크롤이 내려가도 상단 네비게이션 이동이 용이하도록
Fixed Menu를 사용하였습니다.

조명 판매사이트에 맞게 메인 컬러를 밝은 색인 화이트
계열의 컬러로 사용하였으며, 포인트 컬러로 블루 계열
의 컬러를 사용하였습니다.
방문자가 상품 이미지에 집중할 수 있도록 폰트 사이즈,
배경색, 글자색을 절제하여 사용하였습니다.
